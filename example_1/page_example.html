Can you parse me?

We use different ways to parse.

  • can extract as 'soup' attribute 'soup.tag'
  • can use 'soupobj.string' or 'soupobj.tex' to extract text
  • to get an attribute of a tag use 'soupobj['attr_name']'

  • Main methods are: soup.find(), soup.find_all(), soup.select(), soup.select_one()
